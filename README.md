# weather-frontend-m2

Proyecto Modulo 2

## Tema
Versión inspirada en Minecraft, usando biomas del Nether.

## Tecnologías usadas
- HTML5
- CSS
- Bootstrap 5 por CDN
- JavaScript básico
- Git y GitHub

## Páginas
- `index.html`: muestra 10 biomas como cards
- `detalle.html`: muestra información ampliada y pronóstico semanal
- `acerca.html`: explicación simple del proyecto

## Funcionalidades
- Listado de 10 localidades/biomas
- Click en botón para ir al detalle
- Pronóstico semanal
- Diseño responsive para celular y escritorio

## Repositorio GitHub
`https://github.com/GuayoEdu/ClimasMinecraft`


## Cómo usar
1. Descargar o clonar el proyecto
2. Abrir `index.html` en el navegador
3. Probar la navegación entre páginas
